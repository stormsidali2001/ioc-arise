This example demonstrates a simple todo list application using IoC Arise with:

- Repository pattern with two implementations (in-memory and JSON file)
- Service layer for business logic
- Simple entity model

## Project Structure

```
minimal-todo/
├── entities/
│   └── Todo.ts              # Todo entity
├── repositories/
│   ├── ITodoRepository.ts    # Repository interface
│   └── InMemoryTodoRepository.ts # In-memory implementation
├── services/
│   ├── ITodoService.ts       # Service interface
│   └── TodoService.ts        # Business logic service
├── ioc.config.json          # IoC configuration
└── container.gen.ts         # Generated container
```

## Key Components

### Todo Entity

The `Todo` entity represents a simple todo item:

```typescript
export interface Todo {
  id: string;
  title: string;
  description: string;
  completed: boolean;
  createdAt: Date;
  updatedAt: Date;
}
```

### Repository Pattern

The repository interface defines data access operations:

```typescript
export interface ITodoRepository {
  findById(id: string): Promise<Todo | null>;
  findAll(): Promise<Todo[]>;
  create(title: string, description: string): Promise<Todo>;
  update(id: string, updates: Partial<Todo>): Promise<Todo | null>;
  delete(id: string): Promise<boolean>;
}
```

### Service Layer

The service layer contains business logic and depends on the repository:

```typescript
export class TodoService implements ITodoService {
  constructor(private todoRepository: ITodoRepository) {}

  async createTodo(data: { title: string; description: string }): Promise<Todo> {
    return await this.todoRepository.create(data.title, data.description);
  }
  // ... other methods
}
```

## Configuration

The `ioc.config.json` file configures IoC Arise:

```json
{
  "source": ".",
  "output": "container.gen.ts"
}
```

## Generated Container Structure

IoC Arise generates a container with a `coreModule` structure:

```typescript
export const container = {
  coreModule: {
    get TodoService(): TodoService { /* ... */ },
    get ITodoService(): TodoService { /* ... */ },
    get InMemoryTodoRepository(): InMemoryTodoRepository { /* ... */ },
    get ITodoRepository(): InMemoryTodoRepository { /* ... */ }
  }
};
```

## Usage

1. **Generate the container:**
   ```bash
   npx @notjustcoders/ioc-arise generate
   ```

2. **Use the container in your application:**
   ```typescript
   import { container } from './container.gen';
   
   const todoService = container.coreModule.ITodoService;
   await todoService.createTodo({ 
     title: 'Learn IoC Arise', 
     description: 'Study dependency injection' 
   });
   ```
